# HeteroSIM

An OMNeT++ framework for  heterogenuous networks simulations


## Installation commands:


git clone --recurse-submodule https://github.com/MounaKaroui/HeteroSIM.git


git submodule update --remote --merge 


## Required packages:


libboost-dev
